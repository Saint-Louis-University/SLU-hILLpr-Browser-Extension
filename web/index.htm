SLU hILLpr Web Browser Extension

Submit Interlibrary Loan Requests Faster

 

 

Copy WorldCat information to an interlibrary loan request form.

After determining a book is not available via the catalog, MOBIUS, or Prospector, find the book on WorldCat, and click the page action icon in your browser's address bar to copy information to a new interlibrary loan request form.

Source code is available on GitHub.
